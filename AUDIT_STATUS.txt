CRITICAL ISSUE #1: MISMATCH FACTOR INVERSION
================================================================================

SEVERITY: 🔴 CRITICAL
IMPACT: All corrected Jsc and PCE values were inverted

PROBLEM:
  Code was dividing by factor instead of multiplying
  Example: factor=0.95 meant to reduce by 5%, but INCREASED by 5.26%

BEFORE FIX:
  jsc_corrected = jsc / factor  ← WRONG
  40 / 0.95 = 42.1  (5% INCREASE instead of decrease!)

AFTER FIX:
  jsc_corrected = jsc * factor  ← CORRECT
  40 * 0.95 = 38.0  (5% DECREASE as intended) ✅

VERIFICATION:
  ✅ Test 1 passed - Mismatch factor calculation correct
  ✅ PCE recalculation verified
  ✅ All devices on specified day affected correctly
